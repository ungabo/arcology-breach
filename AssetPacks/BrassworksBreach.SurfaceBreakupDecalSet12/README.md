# Brassworks Breach Surface Breakup Decal Set 12

Visual-only Unity sidecar package for the v0.1.57 corridor finish layer. Provides transparent quad decals for soot streaks, wet oil trails, damp floor puddle strips, warm light stain overlays, pipe leak grime, brass edge-wear cards, chipped wall grime, floor scuff clusters, corner darkness cards, and rivet rust halos.

## Import Contract

- Unity 2022.3+ text assets using built-in Quad mesh references and built-in Standard transparent materials.
- Albedo PNGs carry alpha; mask PNGs pack metallic in R, occlusion in G, alpha in B, and smoothness in A for Standard fallback or URP-compatible reuse.
- No colliders, rigidbodies, gameplay scripts, lights, reflection probes, audio, animation clips, scenes, custom shaders, FBX, OBJ, Blender files, or external DCC output.
- Prefabs are placement helpers only; rotate, scale, duplicate, and nudge slightly off surfaces after import.

## Contents

- 20 transparent decal materials under Runtime/Materials.
- 20 visual-only quad prefabs under Runtime/Prefabs.
- 60 generated PNG texture maps under Runtime/Textures.
- Manifest, contact sheet, and concept renders for immediate review.
