# Mechanical Enemy Preview Scene

Use `Brassworks Breach/Sidecars/Mechanical Enemies/Render v0.1.37 Preview PNGs` after importing this package into a Unity project. The menu creates a temporary preview scene, instantiates all five generated prefabs, lights them with amber/cyan foundry lighting, and writes PNG proof renders to:

`Documentation/ConceptRenders/V0_1_37_MechanicalEnemiesSidecar`

The preview scene is intentionally generated instead of checked in so the package stays portable and does not require main-project scene or build-setting changes.
