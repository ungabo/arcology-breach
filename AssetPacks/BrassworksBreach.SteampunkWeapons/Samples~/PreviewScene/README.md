# Preview Scene Notes

This sample folder intentionally contains instructions rather than a checked-in scene. The package preview renderer creates a temporary empty scene, instantiates each generated prefab, lights it with warm brassworks-style key and rim lights, renders PNG files, and clears the scene.

Recommended quarantine workflow:

1. Import this package into a throwaway Unity project.
2. Run `Brassworks Breach/Sidecar Packs/Generate Steampunk Weapon Pack v0.1.37`.
3. Run `Brassworks Breach/Sidecar Packs/Render Steampunk Weapon Previews v0.1.37`.
4. Review the PNGs in `Documentation/ConceptRenders/V0_1_37_SteampunkWeaponsSidecar`.
5. Import into the primary game only after the acceptance report is updated with the render result.
