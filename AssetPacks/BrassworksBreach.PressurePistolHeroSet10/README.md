# Pressure Pistol Hero Set 10

Unity-only visual sidecar package for a component-first pressure pistol hero weapon.

The package contains isolated prefabs for the coil pack, pressure gauge, barrel/tank, muzzle, wood/leather grip, fasteners, and one candidate assembly. Assets are visual-only: no colliders, rigidbodies, gameplay logic, animation controllers, audio, or scene dependencies.

Preview PNGs and the contact sheet are generated to `Documentation/ConceptRenders/V0_1_55_PressurePistolHeroSet10`.

## Contents

- `Runtime/Prefabs`: isolated component prefabs plus candidate assembly.
- `Runtime/Materials`: aged brass, heat-stained copper, blackened iron, dial enamel, amber glass, walnut wood, and worn leather materials.
- `Runtime/Textures`: procedural albedo, normal, and mask textures used by the materials.
- `Runtime/Meshes`: reusable Unity mesh assets generated by the package builder.
- `Runtime/Metadata`: normalized manifest and validation metadata.
- `Documentation~/Manifest`: import-facing copy of the normalized manifest.

## Rebuild

Open the temporary package build project if needed, then run:

`Brassworks Breach/Sidecar Packs/Pressure Pistol Hero Set 10/Generate Assets And Renders`
