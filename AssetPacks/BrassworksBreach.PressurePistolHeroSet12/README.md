# Pressure Pistol Hero Set 12

Unity-only high-fidelity visual sidecar package for the Brassworks Breach first-person pressure pistol hero weapon.

The package contains isolated component prefabs for the wooden/leather grip, brass trigger guard, pressure cylinder/barrel, copper coil array, pressure gauge, side valve wheels, muzzle crown/cog brake, leather glove hand proxy, and the assembled FPS hero pistol. Assets are visual-only: no colliders, rigidbodies, gameplay scripts, animation controllers, audio, or scene dependencies are saved in runtime prefabs.

Preview PNGs are generated to `Documentation/ConceptRenders/V0_1_57_PressurePistolHeroSet12`.

## Contents

- `Runtime/Prefabs`: isolated component prefabs plus the full first-person hero pistol.
- `Runtime/Materials`: aged brass, oxidized copper, dark blued iron, warm amber glass, worn walnut, worn leather, black grime/oil, ivory gauge enamel, and red danger enamel.
- `Runtime/Textures`: procedural PNG albedo, normal, and mask maps used by the materials.
- `Runtime/Meshes`: reusable Unity mesh assets generated by the package builder.
- `Runtime/Metadata`: normalized manifest and validation summary.
- `Documentation~/Manifest`: import-facing copy of the normalized manifest.

## Rebuild

Run the package generator from Unity:

`Brassworks Breach/Sidecar Packs/Pressure Pistol Hero Set 12/Generate Assets And Renders`
