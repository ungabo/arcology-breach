# Brassworks Breach Steam Atmosphere VFX Set 10

Unity-only visual sidecar candidate for the steampunk corridor north-star look.

This package contains visual-only transparent card prefabs, materials, and alpha textures for gaslight haze, steam, mist, wet glints, soot overlays, light shafts, and heat-shimmer proxy planes. It does not contain gameplay scripts, colliders, audio, or scene changes.

Generated package assets live under `Runtime/`. Review PNGs live under `Documentation~/Previews` and the project-level concept render folder.
