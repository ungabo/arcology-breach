# Changelog

## 0.1.40-p001

- Added isolated UPM-style sidecar package for Mechanical Enemy Visual Set 01.
- Added Unity editor generator for thirteen visual-only enemy candidates across five mechanical enemy families.
- Added fifteen generated materials covering brass, blackened iron, copper, leather bellows, glass lenses, warning lamps, cyan pressure tells, furnace heat, soot, hazard paint, rivet wear, ceramic, hot iron, patina pipe, and silhouette ghosting.
- Added eight reusable procedural mesh accents for saw blades, lance tips, shield plates, bellows ribs, gear halos, overseer fins, pressure gauge faces, and drill bits.
- Added Unity preview renderer targeting `Documentation/ConceptRenders/V0_1_40_MechanicalEnemyVisualSet01`.
- Added package-local manifest and package-local validation project for isolated generation without editing the main project package manifest.
