# Brassworks Breach Room Setpiece Kit 04

Isolated Unity sidecar package for visual-only room-scale steampunk setpieces. The package is intended to give later primary-lane levels larger room identities instead of corridor-only geometry.

Generated content target:

- At least 24 visual-only prefabs across ten room setpiece families.
- At least 14 Unity materials.
- At least 6 reusable generated mesh assets.
- Runtime catalog JSON and sidecar manifest JSON.
- Unity proof preview PNGs written to `Documentation/ConceptRenders/V0_1_45_RoomSetpieceKit04`.

No generated prefab should include gameplay scripts, colliders, rigidbodies, or autonomous audio.

Editor menu:

- `Brassworks/Sidecars/Room Setpiece Kit 04 v0.1.45/Generate Package Assets`
- `Brassworks/Sidecars/Room Setpiece Kit 04 v0.1.45/Render Preview PNGs`
- `Brassworks/Sidecars/Room Setpiece Kit 04 v0.1.45/Generate and Render Preview`
