# Weapon Props Set 02 Preview Notes

Import this package into an isolated Unity project, then run:

- `Brassworks Breach/Sidecar Packs/Generate Weapon Props Set 02 v0.1.40`
- `Brassworks Breach/Sidecar Packs/Render Weapon Props Set 02 Previews v0.1.40`

The render pass creates individual previews and `BB_WPS02_ContactSheet.png` under `Documentation/ConceptRenders/V0_1_40_WeaponPropsSet02` when the package is located at `AssetPacks/BrassworksBreach.WeaponPropsSet02`.
