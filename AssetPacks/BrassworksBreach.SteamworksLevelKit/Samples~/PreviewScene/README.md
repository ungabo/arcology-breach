# Preview Scene Notes

This sample folder is intentionally documentation-only for `v0.1.37-p001`.

To produce Unity proof renders:

1. Import this package into a sidecar or quarantine Unity project.
2. Run `Brassworks/Sidecars/Steamworks Level Kit v0.1.37/Generate Package Assets`.
3. Run `Brassworks/Sidecars/Steamworks Level Kit v0.1.37/Render Preview PNGs`.

The render command creates temporary in-scene preview stages at editor time and writes PNG files to:

`Documentation/ConceptRenders/V0_1_37_SteamworksLevelKitSidecar/`

No sample scene asset is included yet because the package is meant to stay small and avoid scene/pipeline settings drift.
