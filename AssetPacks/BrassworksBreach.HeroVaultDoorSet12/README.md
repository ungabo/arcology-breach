# Brassworks Breach Hero Vault Door Set 12

Unity-only sidecar asset family for a high-fidelity brass-and-iron hero pressure vault door. The package is intentionally isolated from playable scenes and contains generated runtime meshes, prefabs, materials, procedural texture maps, metadata, and visual reference renders.

## Runtime Prefabs

- `HVDS12_CircularVaultDoorSlab.prefab`
- `HVDS12_GearWheelLockFace.prefab`
- `HVDS12_RadialBraces.prefab`
- `HVDS12_HingePistonAssemblies.prefab`
- `HVDS12_RivetedArchFrame.prefab`
- `HVDS12_PressureGaugeCluster.prefab`
- `HVDS12_BrassTrimRings.prefab`
- `HVDS12_PipeCouplers.prefab`
- `HVDS12_HeroVaultDoor_Assembly.prefab`

## Notes

All geometry is generated in Unity from code-created meshes and primitive assemblies. No Blender, external DCC files, or game scene integrations are used.
