# Mechanical Enemy Detail Set 12 Documentation

This package is an isolated Unity-only sidecar for mechanical enemy surface-detail lookdev.

## Contents

- Runtime materials and generated PNG maps for aged brass, dark oil-stained iron, worn copper, glowing amber glass, black rubberized hose, sharp saw metal, ivory gauge enamel, soot/oil, and red enamel.
- Runtime mesh assets generated by Unity editor code: cylinders, torus rims, flywheel rims, saw blade, claw blade, hose runs, rivet domes, armor plates, and gauge needles.
- Runtime prefabs for torso plates, glowing head, flywheel shoulder, piston forearm, saw/claw tool, chest gauge, cable/hose bundle, leg joint armor, an assembled bust concept, and a material palette board.
- Runtime metadata manifest and catalog JSON files under `Runtime/Metadata`.

## Integration Notes

The runtime prefabs are visual-only. They intentionally contain no colliders, lights, cameras, gameplay scripts, animation controllers, audio, or scene references. Named `SOCKET_` transforms are included where future rigging or gameplay attachment points are expected.
