# Brassworks Breach - Concept Replica Pipe Bundle Set 13

Unity-only visual sidecar package for a focused pipe-bundle concept replica proof. It provides reusable prefabs for straight pipes, elbow pipes, flanges/collars, wall brackets, valves, dark wet masonry backing, and one assembled pipe-bundle module.

The package is visual-only. Runtime prefabs intentionally contain MeshFilter, MeshRenderer, and Transform components only.
