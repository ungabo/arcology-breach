# Brassworks Breach Steam Lighting Fixture Set 12

Unity-only sidecar asset family for corridor lighting and fixture dressing. The package is generated procedurally from the editor builder in `Editor/SLF12Generator.cs`; no Blender, external DCC, or shared scene integration is required.

Primary generated runtime content:

- Seven fixture prefabs: wall gaslight cage, hanging lantern, vertical inspection lamp, pressure gauge lamp mount, pipe bracket with glow glass, brass switch box, and soot halo decal plane.
- Shared procedural mesh library for cages, collars, rods, rivets, lamps, gauge faces, switch parts, wall plates, and decal planes.
- Procedural PNG material maps for aged brass, oxidized copper, blackened iron, warm emissive glass, soot/grime, wet reflective surfaces, gauge enamel, and porcelain.
- Package metadata and sidecar QA/planning docs.

This pack is visual-only. It intentionally avoids changes to gameplay scenes, shared project manifests, status documents, or main-lane content.
