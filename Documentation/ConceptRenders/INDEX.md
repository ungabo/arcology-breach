# Brassworks Breach Concept Render Index

This folder is a side-agent-only concept/review lane. Files here are outside Unity build assets and should not be imported into gameplay scenes or prefabs unless a main asset pass explicitly promotes them later.

Current state: staged materials, ModularKit OBJ pieces, enemy OBJ blockouts, weapon/prop OBJ blockouts, and HighFidelityLookdev Batch01 OBJ/material placeholders are available. Batch01 was visually rejected as not close enough to the north-star sheet. Recovery03 failed because it was not a convincing Unity/in-engine 3D material proof. Recovery04 proved the Unity render pipeline but was rejected visually as a full-gun target. Recovery05 switched to component-first Unity proofs. Recovery06 improves pressure-pistol component density, but still fails final-art/full-gun promotion because grip/hand quality and composition are not ready. Environment Recovery01 is a useful Unity corridor/material lookdev pass, but production fails until authored modular meshes, collision, LODs, and lighting workflows replace the temporary primitive proof.
