## Visual Review Targets

- Corridors should read as soot-dark, brass-and-copper steamworks infrastructure, not clean sci-fi.
- Door frames should preserve a readable `2.4m x 2.6m` aperture.
- Vault doors, boiler alcoves, gauge walls, and valve consoles should make objectives and locked routes readable from first-person approach distance.
- Pipes, rivets, and railings should add steampunk density without choking combat lanes.
- Smoke anchors should mark emitter sockets only; actual steam volume remains a gameplay/VFX integration task.
