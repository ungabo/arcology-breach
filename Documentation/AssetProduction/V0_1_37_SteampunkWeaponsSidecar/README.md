# v0.1.37 Steampunk Weapons Sidecar

This bundle creates the first dedicated UPM-style asset-pack lane for Brassworks Breach weapon and hand-prop lookdev. It is isolated from the main game project and should be validated through quarantine import before promotion.

## Package Root

`AssetPacks/BrassworksBreach.SteampunkWeapons`

## Scope

- Pressure pistol core
- Visible copper coil assembly
- Brass dial/gauge assembly
- Leather grip module
- Pressure cartridge pickup
- Ammo cabinet shell
- Wall weapon display
- Unity-only preview render generator

## Style Rule

The pack should read as steampunk first: pressure mechanics, brass instrumentation, oxidized copper coils, black iron, dark wood, worn leather, amber glass, and oily industrial stone. Cyberpunk neon and clean sci-fi surfaces are out of scope for this lane.
