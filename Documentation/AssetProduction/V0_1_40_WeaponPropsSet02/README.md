# V0.1.40 Weapon Props Set 02

## Scope

Weapon Props Set 02 is a new sidecar-only package at `AssetPacks/BrassworksBreach.WeaponPropsSet02`. It expands first-person/world weapon and interactable prop visual candidates for Brassworks Breach without touching gameplay scenes, shared package roots, or the existing SteampunkWeapons package.

## Generated Counts

- Prefabs: 16
- Materials: 12
- Reusable meshes: 4
- Preview renders: 16 individual PNGs plus 1 contact sheet
- Runtime scripts: 1 identity component
- Editor scripts: 1 generator and 1 preview renderer

## Visual Intent

The set leans into dense steampunk FPS readability: pressure pistol component variants, scattergun body variants, amber pressure cartridges, wall weapon storage, an ammo cabinet shell, gear-key housing, a pressure-cell canister, tuning dials and gauges, and repair-tool clutter. Subparts are named for later promotion, replacement, sockets, and LOD work.

## Non-Integration Rule

This batch is package-candidate art only. It intentionally includes no damage logic, pickup logic, inventory references, animation sockets, generated gameplay scenes, or edits to `Packages/manifest.json`.
