## Rename Procedure

1. Pick the new title and executable-safe stem.
2. Update `GameBranding.cs`.
3. Update Unity product name in `ProjectSettings/ProjectSettings.asset`.
4. Replace documentation references.
5. Rename the GitHub repository if desired.
6. Run scene rebuild, editor smoke, Windows build, and packaged smokes.
7. Update `BUILD_STATUS.md` and this file.
8. Commit and push.

## Versioned Build Numbering

When a meaningful local test point is reached:

1. Increment `GameBranding.BuildVersion`.
2. Rebuild the scene.
3. Build Windows.
4. Run packaged smokes.
5. Update `BUILD_STATUS.md`, `SESSION_LOG.md`, and active tracking docs.
6. Commit and push.

Build executable pattern:

`Builds/Windows/<version>/BrassworksBreach_<version>.exe`
