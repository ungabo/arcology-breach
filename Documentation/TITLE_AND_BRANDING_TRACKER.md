## Rename Procedure

1. Pick the new title and executable-safe stem.
2. Update `GameBranding.cs`.
3. Update Unity product name in `ProjectSettings/ProjectSettings.asset`.
4. Replace documentation references.
5. Rename the GitHub repository if desired.
6. Run scene rebuild, editor smoke, Windows build, and runtime smoke.
7. Update `HANDOFF.md`, `BUILD_STATUS.md`, and this file.
8. Commit and push.

## Checkpoint Build Versioning

When a meaningful local test point is reached:

1. Increment `GameBranding.CheckpointVersion`.
2. Rebuild the scene.
3. Build Windows.
4. Run runtime smoke.
5. Update `BUILD_STATUS.md`, `HANDOFF.md`, and `SESSION_LOG.md`.
6. Commit and push.

Checkpoint executable pattern:

`Builds/Windows/<version>/ArcologyBreach_<version>.exe`
