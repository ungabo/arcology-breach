## Recommended Import Methodology

1. Create a separate Unity sandbox project for asset-pack inspection.
2. Import one package at a time and record render pipeline, dependency, license, folder layout, mesh scale, material count, shader usage, prefab complexity, and performance notes.
3. Export only approved, curated assets into a clean staging folder, never directly into `Assets/_Project`.
4. Rebuild final Brassworks assets with project-owned naming, prefab sockets, LODs, colliders, material variants, and platform tiers.
5. For any direct third-party asset use, keep an attribution/license note in the production asset catalog and make sure final repo packaging does not redistribute anything outside the permitted license.
