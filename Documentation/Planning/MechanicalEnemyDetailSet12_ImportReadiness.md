# Mechanical Enemy Detail Set 12 Import Readiness

The package can be added as a local Unity package from `AssetPacks/BrassworksBreach.MechanicalEnemyDetailSet12`.

## Notes

- Visual-only prefabs contain MeshFilter and MeshRenderer components plus empty SOCKET transforms.
- No colliders, lights, cameras, scripts, animation controllers, AI, audio, or scene dependencies are included in runtime prefabs.
- Generated preview PNGs are mirrored to package `Runtime/Previews` and documentation concept renders.
- Future integration should create LODs, batching, damage sockets, animator bindings, and collision separately.
